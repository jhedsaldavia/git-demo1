pwd - print working directory
      display your current directory location

cd [lol] - change directory

mkdir [folder name] - make a new folder in the current directory

ls - displays the contents of your currently directory

touch [filename] - creates a new file in the current directory

rm [filename] - deleting/removing a file

repository - like a folder/container that stores file/folders related to a topic. It keep track of the changes made over time

2 types of repository
Local - version of repo that's stored on your own local machine
Remote - version of repo that is stored on a server/cloud based platform

git init - initialize a folder

git config --global user.name "username"

git config --global user.email "email@email.com"

git add . 

git add .

		- adds all modified files in your current directory to the staging area

 

*	git add "filename"

		- adds a specific file to the staging area

 

*	git commit -m "message"

		- saves the changes that you've made to the files in the staging area to the repository. The message describes what changes were made in this commit. 

 

*	git status 

		- displays the current status of your file/folder
